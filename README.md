# Majestique Landmarks — Homepage

## Project Structure

```
majestique-website/
├── index.html                  ← Main homepage (open this in browser)
├── css/
│   └── style.css               ← All styles
├── js/
│   └── main.js                 ← All JavaScript
└── assets/
    ├── images/
    │   ├── logo.png            ← Majestique Landmarks logo
    │   ├── aravali-1.webp      ← Aravali project image (primary)
    │   ├── aravali-2.webp      ← Aravali project image (alternate)
    │   └── beyond4walls.jpg    ← #Beyond4Walls family photo
    └── video/
        └── hero.mp4            ← Hero background video

```

## How to Run

1. **Locally** — Open `index.html` directly in any modern browser (Chrome, Firefox, Edge, Safari)
2. **Via Live Server** — Use VS Code Live Server extension on port 5500
3. **Deploy** — Upload the entire `majestique-website/` folder to any web host
